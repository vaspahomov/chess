# Шахматы 

# Реализовано 
* Класс GameObject 
* Класс Figure для наследования из него фигур
* Класс Field, где хранятся клетки Cell
* Классы для фигур
* Отрисовка поля и фгиур
* Перемещение фигур по правилам шахмат
* Игра на 2 игроков
* Реализация игровых позиций (Шах)
